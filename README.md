# AI-Assisted Interview Preparation - Flutter App

This repository contains the **Flutter mobile client** for the AI-Assisted Interview Preparation App.

Users can:

- Log in / register
- Submit answers via text or audio
- Receive AI-generated feedback
- Track progress and personalized recommendations
- View dashboard analytics

The app communicates with the backend API: [ai-interview-backend](https://github.com/your-username/ai-interview-backend)

## Tech Stack

- Flutter 3+
- Dart
- Riverpod / Provider for state management
- Dio for HTTP requests
- Shared Preferences for local storage
- GoRouter for navigation

## Getting Started

1. Clone the repo  
2. Install Flutter SDK 3+  
3. Run `flutter pub get`  
4. Configure API base URL  
5. Run app: `flutter run`
